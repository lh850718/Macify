# Flutter Content Assets

This directory is generated by `npm run content:build` from the shared `content/` source files.

Add it to a Flutter app with:

```yaml
flutter:
  assets:
    - assets/content/
```

The app should read `content-manifest.json` first, then load `config.json`, `videos.json`, `ambient-tracks.json`, `ambient-rules.json`, and `video-audio-mixes.json`.
